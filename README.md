## Sobre

Sou Leonardo Ramos, profissional em desenvolvimento na área de tecnologia com foco em **redes, infraestrutura de TI e cibersegurança**.

Este GitHub será meu portfólio prático: um espaço para publicar laboratórios, estudos, scripts, anotações técnicas e projetos voltados para Cybersecurity, especialmente em Blue Team, SIEM, análise de logs, hardening e investigação de eventos.

## Foco Técnico

- Redes, infraestrutura de TI e segurança da informação
- Blue Team, SIEM, análise de logs e resposta a incidentes
- Linux, linha de comando, Git, GitHub e automações simples
- Laboratórios de vulnerabilidades, hardening e documentação técnica
- Projetos práticos que demonstrem evolução, método e clareza
